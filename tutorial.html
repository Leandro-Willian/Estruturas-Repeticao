Tutorial: Estruturas de Repetição em JavaScript

Introdução

As estruturas de repetição são fundamentais na lógica de programação para executar tarefas repetitivas de forma eficiente. No JavaScript, temos três estruturas principais: for, while e do...while. Cada uma delas é usada para diferentes cenários.

Estrutura for

O loop for é usado quando sabemos exatamente quantas vezes um bloco de código deve ser repetido.

for (let i = 1; i <= 10; i++) {
    console.log(i);
}

Exemplo: Contar de 1 a 10.

let soma = 0;
for (let i = 1; i <= 5; i++) {
    soma += i;
    console.log(soma);
}

Exemplo: Soma acumulada dos números de 1 a 5.

const frutas = ["Maçã", "Banana", "Laranja"];
for (let i = 0; i < frutas.length; i++) {
    console.log(frutas[i]);
}

Exemplo: Listar elementos de um array.

for (let i = 2; i <= 20; i += 2) {
    console.log(i);
}

Exemplo: Exibir os números pares entre 2 e 20.

for (let i = 10; i >= 0; i--) {
    console.log(i);
}

Exemplo: Contagem regressiva de 10 a 0.

Estrutura while

O loop while é usado quando a repetição depende de uma condição ser verdadeira.

let contador = 1;
while (contador <= 5) {
    console.log(contador);
    contador++;
}

Exemplo: Contar de 1 a 5.

let numero = 2;
while (numero <= 32) {
    console.log(numero);
    numero *= 2;
}

Exemplo: Exibir potências de 2 até 32.

let soma = 0;
let i = 1;
while (i <= 10) {
    soma += i;
    i++;
}
console.log(soma);

Exemplo: Soma dos números de 1 a 10.

let numero = 10;
while (numero >= 1) {
    console.log(numero);
    numero--;
}

Exemplo: Contagem regressiva de 10 a 1.

let valor = 1;
while (valor < 100) {
    console.log(valor);
    valor *= 3;
}

Exemplo: Multiplicar um número por 3 até ser maior que 100.

Estrutura do...while

O loop do...while executa o bloco de código pelo menos uma vez antes de verificar a condição.

let contador = 1;
do {
    console.log(contador);
    contador++;
} while (contador <= 5);

Exemplo: Contar de 1 a 5.

let resposta;
do {
    resposta = prompt("Digite 'sim' para continuar ou 'não' para sair:");
} while (resposta.toLowerCase() !== 'não');

Exemplo: Solicitar ao usuário uma confirmação até digitar 'não'.

let senhaCorreta = "1234";
let senha;
do {
    senha = prompt("Digite a senha:");
} while (senha !== senhaCorreta);
console.log("Acesso permitido.");

Exemplo: Solicitar a senha até estar correta.

let numero;
do {
    numero = Math.floor(Math.random() * 10);
    console.log(numero);
} while (numero !== 7);

Exemplo: Gerar números aleatórios até obter 7.

let total = 0;
do {
    let entrada = prompt("Digite um número para somar (ou 0 para sair):");
    total += parseInt(entrada);
} while (entrada !== "0");
console.log("Total acumulado:", total);

Exemplo: Somar números até que o usuário insira 0.

Exercícios Práticos

Abaixo estão exercícios propostos para que você pratique o que aprendeu sobre as estruturas de repetição em JavaScript. Tente resolvê-los no editor de código de sua preferência.

Exercícios com for

  • Crie um programa que exiba os números ímpares de 1 a 20.

Exercícios com while
